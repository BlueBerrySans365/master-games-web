# Master Games Website

## Project setup
```
npm install
```
```
npm install pm2 -g
```
```
npm install -g serve
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Build and deploy
```
npm run build
```
```
pm2 serve -s -l 443 dist
```

All pm2 commands are [Here](https://www.npmjs.com/package/pm2).
